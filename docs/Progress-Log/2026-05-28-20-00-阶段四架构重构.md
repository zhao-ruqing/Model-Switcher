# 2026-05-28 20:00 — 阶段四架构重构

## 已开发功能总结

- 支持 Claude Code 和 OpenCode 双工具的配置管理与一键切换
- 提供配置列表的增删改查及当前激活状态标记
- 支持 DeepSeek、MiMo、OpenAI、Gemini、GLM、KIMI、Mistral、Groq 等主流 Provider 预设
- 配置导入导出功能，方便备份与迁移
- API Key 连通性测试，验证 URL、Key 及对话可用性
- 本地配置文件一致性校验，确保切换结果正确落盘
- 分步配置向导，引导新手完成首次配置
- 暗黑明亮主题切换，LocalStorage 记忆偏好
- 全局 Lucide 图标库替换原有 Emoji，UI 风格统一
- 插件式 ProviderAdapter 架构，新增工具只需添加适配器文件
- Provider 预设数据外置到 presets.json，新增 Provider 只需修改配置文件
